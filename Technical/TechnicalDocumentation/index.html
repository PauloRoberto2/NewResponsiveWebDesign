Introdução

Bem-vindo à documentação da linguagem de programação C#! C# é uma linguagem de programação moderna, orientada a objetos e fortemente tipada, projetada pela Microsoft para o desenvolvimento de aplicativos para a plataforma .NET. Esta linguagem combina características de várias outras linguagens populares, incluindo C++, Java e Visual Basic, tornando-a fácil de aprender para programadores com experiência em outras linguagens.

A documentação aqui apresentada tem como objetivo fornecer aos programadores um guia para a linguagem C#, incluindo sua sintaxe, principais recursos e funcionalidades, além de dicas úteis para o desenvolvimento de aplicativos. Se você é um iniciante em programação ou um desenvolvedor experiente procurando se atualizar, esta documentação irá ajudá-lo a dominar a linguagem C# e desenvolver aplicativos de qualidade.

Ao longo deste documento, você aprenderá sobre as principais características da linguagem C#. Além disso, você também terá acesso a exemplos de código para ilustrar os conceitos apresentados, permitindo que você pratique e aprimore suas habilidades de programação. Então, vamos começar a explorar a linguagem de programação C# e descobrir como ela pode ajudá-lo a criar aplicativos poderosos e eficientes para a plataforma .NET!

Arquitetura do .NET

O C# é uma linguagem de programação que é executada no .NET, um sistema de execução virtual chamado CLR e um conjunto de bibliotecas de classes. O CLR é uma implementação da CLI, um padrão internacional que fornece uma base para criar ambientes de execução e desenvolvimento onde as linguagens e bibliotecas funcionam em conjunto perfeitamente.

O código C# é compilado em IL, uma linguagem intermediária que está em conformidade com a especificação da CLI. O código IL é armazenado em um assembly, normalmente com extensão .dll, que contém um manifesto que fornece informações sobre os tipos, versão e cultura. Quando o programa é executado, o assembly é carregado no CLR e a compilação JIT é usada para converter o código IL em instruções nativas da máquina.

O .NET suporta interoperabilidade de linguagem, onde o código IL produzido pelo compilador C# pode interagir com o código produzido por outras linguagens do .NET, como F#, Visual Basic e C++. O .NET também inclui uma ampla variedade de bibliotecas organizadas em namespaces que fornecem funcionalidades úteis para diferentes cargas de trabalho, desde entrada e saída de arquivos até a manipulação de cadeias de caracteres, análise de XML e estruturas de aplicativos Web para controles de Windows Forms. As bibliotecas são amplamente utilizadas em aplicativos C# para lidar com tarefas comuns de "conexão".

Tipos e variáveis

Em C#, um tipo define a estrutura e o comportamento de um dado. Há dois tipos: tipos de referência (referenciando objetos) e tipos de valor (diretamente contendo seus dados). Os identificadores são nomes de variáveis e podem ser palavras reservadas se prefixados por "@". Os tipos de valor são divididos em tipos simples, enumeração, struct, anulável e tupla. Os tipos de referência são divididos em classe, interface, matriz e delegados.

A estrutura de tópicos a seguir fornece uma visão geral do sistema de tipos do C#.

  • Tipos de valor
    • Tipos simples
      • Integral com sinal: sbyte, short, int, long
      • Integral sem sinal: byte, ushort, uint, ulong
      • Caracteres Unicode: char, ue representa uma unidade de código UTF-16
      • Ponto flutuante binário de IEEE: floatdouble
      • Ponto flutuante decimal de alta precisão: decimal
      • Booliano: bool, que representa valores boolianos - valores que são true ou false
    • Tipos de referência
